# Song filter quick reference

| Prefix | Usage                         |
| ------ | ----------------------------- |
| -      | Match in Title, Singer and ID |
| `I:`   | Match exactly ID              |
| `X:`   | Match exactly index           |
| `N:`   | Match in Title                |
| `S:`   | Match in Singer               |
| `C:`   | Match in Album                |
| `A:`   | Match in Author               |
| `LA:`  | Match in Lyrics Author        |
| `MA:`  | Match in Music Author         |
| `M:`   | Match exactly rating          |
| `ML:`  | Match max rating              |
| `MG:`  | Match min rating              |
